# Simple To-Do List (Laravel)

Mini project untuk take-home test magang di PT Bejana Investidata Globalindo.

## Fitur
- Register & Login
- Tambah tugas
- Tandai selesai / belum
- Hapus tugas
- Setiap user hanya melihat task miliknya sendiri
